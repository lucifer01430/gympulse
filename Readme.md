## 🚀 Overview

GymPulse allows gym owners to:
* Manage gym members
* Track attendance
* Record payments and dues
* Generate professional invoices
* View reports and analytics

All of this works directly in the browser **without any backend**.

---

## ⚙️ Tech Stack

* HTML5
* CSS3
* JavaScript (Vanilla JS)
* Bootstrap

---

## 🧠 Features

### 👤 Member Management
* Add / Edit / Delete members
* Track membership details

### 📅 Attendance Tracking
* Daily attendance marking

### 💰 Payments & Dues
* Record payments & track pending dues

### 🧾 Invoice System
* Generate invoices, Download PDF, Print-ready

### 📊 Dashboard & Reports
* Total members, Revenue insights, Attendance reports
